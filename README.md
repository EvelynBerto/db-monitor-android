# DB Monitor Android 🎙️📊

Um painel de controle em tempo real para monitoramento de níveis de ruído (decibéis), desenvolvido com **Jetpack Compose** e **Clean Architecture**.

## ✨ Funcionalidades
- **Monitoramento em Tempo Real:** Captura de áudio e conversão para dB.
- **Gráfico Dinâmico:** Visualização da oscilação sonora com linhas de referência (40, 80, 100 dB).
- **Alertas Visuais:** Mudança de cor dinâmica (Verde/Amarelo/Vermelho) conforme o nível de ruído.
- **Histórico Local:** Persistência de picos de ruído acima de 85dB utilizando **Room Database**.
- **Injeção de Dependência:** Gerenciado com **Koin**.

## 🛠️ Tecnologias
- Kotlin 2.0.21
- Jetpack Compose (Material 3)
- Coroutines & Flow
- Room DB
- Koin DI
- KSP (Kotlin Symbol Processing)
